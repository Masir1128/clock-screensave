# 时钟屏保 · Clock Screensaver

A beautiful fullscreen clock screensaver for macOS, built with Electron.

![preview](https://raw.githubusercontent.com/Masir1128/clock-screensave/main/assets/preview.png)

## Features

- **Fullscreen clock** — large HH:MM:SS display with animated gradient text
- **Starfield background** — hundreds of gently drifting, twinkling stars
- **8 gradient presets** — Aurora, Sunset, Ocean, Forest, Gold, Neon, Rose, White
- **Custom color picker** — pick any solid color from the rainbow swatch
- **Auto-show on idle** — appears automatically after 30 s / 1 min / 2 min / 5 min (configurable)
- **Lock-screen integration** — shows when the screen locks, hides on unlock
- **Anti-burn-in drift** — content shifts every 60 s to protect OLED displays
- **Color persistence** — remembers your last color selection across restarts
- **macOS menu bar app** — lives quietly in the menu bar (⏰), zero Dock clutter

## Installation (pre-built)

1. Download the latest `.dmg` from [Releases](https://github.com/Masir1128/clock-screensave/releases)
2. Open the DMG and drag **时钟屏保** into your Applications folder
3. Launch it — the ⏰ icon appears in your menu bar

> **First launch:** macOS may show a security prompt. Go to  
> **System Settings → Privacy & Security** and click **Open Anyway**.

## Usage

| Action | How |
|--------|-----|
| Show / hide clock | Click the ⏰ menu bar icon |
| Close the clock | Press **ESC** or click **✕** in the bottom control bar |
| Change color / gradient | Click any dot in the bottom control bar |
| Set idle timeout | Right-click ⏰ → **无操作后自动显示** |
| Quit | Right-click ⏰ → **退出** |

## Setting as a Screensaver

macOS does not natively support third-party `.app` files as screensavers.  
The recommended approach is to use **Lungo** or **Almighty** to keep the display on, and let this app trigger on idle.  

Alternatively, use **Automator** or a login item to launch on startup:

1. **System Settings → General → Login Items**
2. Click **+** and add `时钟屏保.app`

The app auto-shows on the configured idle timeout and on screen lock — effectively acting as a screensaver without requiring a `.saver` bundle.

## Development

### Prerequisites

- macOS 12+
- Node.js 18+

### Setup

```bash
git clone git@github.com:Masir1128/clock-screensave.git
cd clock-screensave
npm install
npm start
```

### Project structure

```
├── main.js        # Electron main process (window, tray, power events)
├── preload.js     # Context bridge (IPC between main ↔ renderer)
├── index.html     # Renderer: clock UI, starfield canvas, color picker
├── scripts/
│   └── gen-icon.js  # Generates assets/icon.png (no extra deps required)
└── assets/
    └── icon.png   # App icon (auto-generated by gen-icon script)
```

### Build

```bash
npm run build
```

This runs `gen-icon` first (creates `assets/icon.png`), then packages a universal DMG for both Apple Silicon and Intel Macs via `electron-builder`.

Output is in the `dist/` folder.
